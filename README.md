Atividade 6 — Calculadora de INSS e IRPF:

Projeto desenvolvido para a disciplina de Programação Orientada a Objetos.  
---------------------------------------------------------------------------------

Objetivo:

A atividade tem como finalidade praticar:

- Criação de **Interfaces**
- Uso de **Herança** (superclasse + subclasses)
- Aplicação de **Polimorfismo** em Java

O projeto implementa um sistema simples de cálculo de tributos (INSS e IRPF) para diferentes tipos de contribuintes.
----------------------------------------------------------------------------------
O programa:

1. Lê dados do usuário (nome, CPF, salário).
2. Permite escolher o tipo de contribuinte:
   - Funcionário CLT  
   - Estagiário  
   - Prestador de Serviço (PJ)
3. Calcula:
   - INSS  
   - IRPF  
   - Salário líquido  
4. Exibe um resumo no console.
